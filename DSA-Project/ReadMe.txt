<------Tic Tac Toe Gam------>
-->Language: C++
-->Algorithm used: Min-Max Algorithm
-->User can never win this game.
-->Min-Max Explained:
Min-Max is an AI based algorithm used in decision making
games for example: chess, tic tac toe etc.
It makes a virtual game tree where it checks all the possible 
moves. This algorithm hasa two sides min and max it finds the
best possible moves for both and at the end returns the best
move for the computer that makes it impossible for the user 
to defeat computer.
